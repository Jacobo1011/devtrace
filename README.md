# DevTrace

**DevTrace** is a modern full-stack system monitoring dashboard built with **Next.js** and **FastAPI**.
It was created to make system metrics easier to visualize and understand through a clean, responsive, and real-time interface.

The project combines a powerful Python backend with a modern React frontend, allowing you to monitor performance data live while exploring a scalable architecture using current web technologies.

Whether you're learning full-stack development, experimenting with monitoring tools, or building the foundation for a larger infrastructure project, DevTrace provides a simple and developer-friendly starting point.

# Features

* Real-time system metrics monitoring
* Interactive and responsive dashboard UI
* Live communication between frontend and backend
* Modern full-stack architecture
* Built with scalable technologies
* Clean and minimal design focused on usability

# Getting Started

## Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

Once the development server starts, open:

```bash
http://localhost:3000
```

to view the dashboard in your browser.

---

## Backend Setup

```bash
cd backend
pip install -r requirements.txt
python -m uvicorn main:app --reload
```

The API will be available at:

```bash
http://localhost:8000
```
